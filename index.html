Política de Privacidade

JSM Lubrificação Inteligente

Última atualização: Julho de 2026.

1. Informações Gerais

O aplicativo JSM Lubrificação Inteligente foi desenvolvido para auxiliar profissionais da manutenção industrial no cálculo de lubrificação de rolamentos e consulta de informações técnicas.

2. Coleta de Dados

Este aplicativo NÃO coleta dados pessoais dos usuários.

Não coletamos:

  • Nome
  • E-mail
  • Telefone
  • Localização
  • Documentos
  • Informações financeiras

3. Compartilhamento de Dados

Nenhum dado é compartilhado com terceiros.

4. Segurança

O aplicativo funciona localmente no dispositivo do usuário, oferecendo maior segurança e privacidade.
